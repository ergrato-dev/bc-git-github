# Ebooks Gratuitos - Week 05: Remote Repositories

Libros electronicos gratuitos para profundizar en repositorios remotos y colaboracion.

---

## Libros Recomendados

### 1. Pro Git (Scott Chacon & Ben Straub)

**Capitulos relevantes:**

- Chapter 2.5: Working with Remotes
- Chapter 3.5: Remote Branches
- Chapter 5: Distributed Git
- Chapter 6.1: Account Setup and Configuration

**Enlace:** [https://git-scm.com/book/en/v2](https://git-scm.com/book/en/v2)

**Idioma:** Ingles y Espanol disponibles

**Formato:** PDF, EPUB, HTML (online gratuito)

---

### 2. Git Pocket Guide (Richard E. Silverman)

**Secciones relevantes:**

- Remote Repositories
- Fetching and Pulling
- Pushing Changes
- Tracking Branches

**Nota:** Disponible en algunas bibliotecas digitales.

---

### 3. Git Notes for Professionals

**Capitulos relevantes:**

- Chapter 10: Working with Remotes
- Chapter 11: Pushing to a remote
- Chapter 12: Fetching
- Chapter 17: Forking workflow

**Enlace:** [https://goalkicker.com/GitBook/](https://goalkicker.com/GitBook/)

**Formato:** PDF gratuito

---

### 4. GitHub Skills

**Modulos relevantes:**

- Introduction to GitHub
- Communicate using Markdown
- GitHub Pages
- First Week on GitHub

**Enlace:** [https://skills.github.com/](https://skills.github.com/)

**Formato:** Cursos interactivos gratuitos

---

## Como Descargar

### Pro Git

```bash
# Clonar repositorio con el libro
git clone https://github.com/progit/progit2.git

# O descargar PDF directamente desde:
# https://github.com/progit/progit2/releases
```

### Git Notes for Professionals

1. Visitar [goalkicker.com/GitBook/](https://goalkicker.com/GitBook/)
2. Click en "Download Git book"
3. PDF gratuito de ~200 paginas
