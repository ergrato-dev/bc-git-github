# Ejercicios Practicos - Week 05: Remote Repositories

## Descripcion

Ejercicios guiados para dominar el trabajo con repositories remotos, sincronizacion y colaboracion.

**Tiempo total:** 2.5 horas

---

## Lista de Ejercicios

| # | Ejercicio | Tema | Duracion | Dificultad |
|---|-----------|------|----------|------------|
| 01 | [Clonar y Configurar Remotes](ejercicio-01-clonar-remotes/) | clone, remote add/remove | 35 min | Principiante |
| 02 | [Push y Pull Basico](ejercicio-02-push-pull/) | push -u, pull, tracking | 35 min | Principiante |
| 03 | [Fetch y Merge Manual](ejercicio-03-fetch-merge/) | fetch, diff, merge manual | 40 min | Intermedio |
| 04 | [Fork Workflow](ejercicio-04-fork-workflow/) | fork, upstream, PR | 40 min | Intermedio |

---

## Progresion de Ejercicios

```
Ejercicio 01          Ejercicio 02          Ejercicio 03          Ejercicio 04
Clonar/Remotes   -->  Push/Pull        -->  Fetch/Merge      -->  Fork Workflow
[35 min]              [35 min]              [40 min]              [40 min]
     |                     |                     |                     |
     v                     v                     v                     v
  clone               push -u              fetch               remote add
  remote -v           pull                 log ..              upstream
  remote add          tracking             merge               sync fork
```

---

## Checklist de Progreso

- [ ] Ejercicio 01: Clonar y Configurar Remotes
- [ ] Ejercicio 02: Push y Pull Basico
- [ ] Ejercicio 03: Fetch y Merge Manual
- [ ] Ejercicio 04: Fork Workflow

---

## Requisitos

Antes de comenzar los ejercicios asegurate de:

- [ ] Tener cuenta de GitHub activa
- [ ] Tener SSH keys configuradas
- [ ] Completar la teoria de la semana
- [ ] Tener Git instalado y configurado

---

## Comandos que Practicaras

| Ejercicio | Comandos Principales |
|-----------|---------------------|
| 01 | `git clone`, `git remote -v`, `git remote add`, `git remote remove` |
| 02 | `git push -u`, `git pull`, `git status`, `git branch -vv` |
| 03 | `git fetch`, `git log main..origin/main`, `git merge origin/main` |
| 04 | `git remote add upstream`, `git fetch upstream`, `git merge upstream/main` |
