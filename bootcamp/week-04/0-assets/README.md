## Estandares de Diseno

- **Formato**: SVG (Scalable Vector Graphics)
- **Tema**: Dark mode (fondo `#0d1117`)
- **Colores**: Solidos, sin degradados
- **Tipografia**: Arial, sans-serif / monospace para codigo

| Color | Hex | Uso |
