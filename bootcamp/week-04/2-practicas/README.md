# Ejercicios Practicos - Week 04

## Descripcion

Ejercicios guiados para dominar la resolucion de merge conflicts.

---

## Objetivos

- Provocar conflictos intencionalmente para entenderlos
- Resolver conflictos de diferentes tipos y complejidades
- Usar herramientas visuales (VS Code, mergetool)
- Manejar situaciones de emergencia (abort, reset)

---

## Lista de Ejercicios

| # | Ejercicio | Descripcion | Duracion |
|---|-----------|-------------|----------|
| 01 | [conflicto-simple](ejercicio-01-conflicto-simple/) | Provocar y resolver tu primer conflicto | 35 min |
| 02 | [multiples-archivos](ejercicio-02-multiples-archivos/) | Conflictos en 3 archivos simultaneos | 40 min |
| 03 | [vscode-mergetool](ejercicio-03-vscode-mergetool/) | Resolver con herramientas visuales | 40 min |
| 04 | [abortar-reintentar](ejercicio-04-abortar-reintentar/) | Cancelar merge y estrategias de reintento | 35 min |

**Tiempo total estimado:** 2.5 horas

---

## Orden Recomendado

1. **Ejercicio 01**: Fundamental - tu primer conflicto
2. **Ejercicio 02**: Escalar a multiples archivos
3. **Ejercicio 03**: Herramientas que facilitan el trabajo
4. **Ejercicio 04**: Manejo de errores y recuperacion

---

## Requisitos

- Git instalado y configurado
- VS Code (para ejercicio 03)
- Terminal disponible
- Completar teoria de Week 04

## Tips para los Ejercicios

1. **Lee todo antes de empezar** - Entiende el objetivo completo
2. **No copies y pegues ciegamente** - Escribe los comandos
3. **Observa los resultados** - Compara con lo esperado
4. **Experimenta** - Prueba variaciones de los comandos
5. **Pregunta si no entiendes** - Es mejor preguntar que avanzar sin comprender
