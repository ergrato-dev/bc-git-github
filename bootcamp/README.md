# 🛠️ Bootcamp Git/GitHub - Trabajo Efectivo

Esta carpeta contiene todo el material de trabajo práctico semanal del bootcamp de 14 semanas.

## 📅 Estructura Semanal

Cada semana sigue una estructura estandarizada:

```
semana-XX/
├── README.md              # Descripción general de la semana
├── RUBRICA_EVALUACION.md  # Criterios de evaluación específicos
├── 1-teoria/              # Lecciones y conceptos teóricos
├── 2-ejercicios/          # Prácticas guiadas paso a paso
├── 3-recursos/            # Material complementario semanal
└── 4-proyecto/            # Proyecto integrador de la semana
    ├── README.md          # Instrucciones del proyecto
    ├── instrucciones.md   # Detalles específicos
    ├── entregables/       # Archivos a entregar
    ├── recursos/          # Material de apoyo
    └── solucion/          # Solución de referencia
```

## 🎯 Fases del Bootcamp

### 📘 Fase 1: GitHub Foundations (Semanas 1-6)

- **Objetivo**: Dominio completo de Git y GitHub básico
- **Certificación**: GitHub Foundations

### 🔧 Fase 2: GitHub Actions (Semanas 7-10)

- **Objetivo**: Automatización CI/CD con GitHub Actions
- **Certificación**: GitHub Actions

### 🛡️ Fase 3: GitHub Advanced Security (Semanas 11-12)

- **Objetivo**: Seguridad y vulnerability management
- **Certificación**: GitHub Advanced Security

### 👥 Fase 4: Administration + Proyecto Final (Semanas 13-14)

- **Objetivo**: Administración enterprise y proyecto integrador
- **Certificación**: GitHub Administration

## 📋 Instrucciones de Uso

1. **Para Instructores**: Cada semana tiene material completo listo para usar
2. **Para Estudiantes**: Seguir la numeración secuencial (1-teoria, 2-ejercicios, etc.)
3. **Evaluación**: Usar las rúbricas específicas de cada semana
